# Simple Banking System in C++

## Overview
This is a simple CLI-based banking management system built in C++. It allows you to:
- Show existing clients
- Add new clients
- Delete a client
- Update client information
- Search for clients by account number

All client records are stored in a `.txt` file using a custom separator.

---

## Features
- File-based data storage (no database needed)
- Safe client deletion using soft delete
- Easy-to-read modular code structure
- Menu-based navigation system

---

## Technologies Used
- C++
- Standard Libraries: `<vector>`, `<fstream>`, `<string>`
